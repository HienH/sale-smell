üî• Product Summary
Sale-Smell is a web AI tool that allows users to upload recorded sales calls and receive intelligent, actionable feedback. By combining speech-to-text transcription (via AssemblyAI) with a local LLM (LLaMA 3 via Ollama), the app provides sales coaching insights, sentiment analysis, and structured call metrics. Users can download a detailed PDF report based on the analysis.

üéØ Goals
Help sales reps improve by analyzing real sales conversations.

Automatically transcribe, summarize, and assess call quality.

Surface coaching insights: tone, objection handling, talk ratio, etc.

Deliver a downloadable coaching report.

Keep the app lightweight and frictionless: no login required for MVP.
